## ADDED Requirements

### Requirement: Technical Support button
The system SHALL display a "Technical Support" button at the bottom of all screens.

#### Scenario: Support button visible
- **WHEN** any screen is displayed
- **THEN** a "Technical Support" button is visible at the bottom

### Requirement: QR code display
When clicked, the Technical Support button SHALL display a QR code linking to a Telegram support URL (https://t.me/+AQgpJzP4t4thMzdi).

#### Scenario: QR code shown
- **WHEN** user clicks "Technical Support" button
- **THEN** a QR code is displayed that links to the Telegram support URL
